About MovieCenter

MovieCenter is a modern movie discovery web application designed to help users search and explore movies with ease. It provides real-time movie details such as ratings, cast, genre, runtime, and plot by integrating the OMDb API. Built using HTML, CSS, and JavaScript, the platform focuses on a clean, responsive, and SaaS-style user interface to ensure a smooth and engaging browsing experience across both desktop and mobile devices. The application also highlights popular movies to help users discover trending content quickly. Performance and usability were prioritized to deliver fast search results and an intuitive user experience.
